# django-project

## Using this project
```
Change directory to .../django-project
```
### Start server
```
python manage.py runserver
```
#### Alternatively change the port by
```
python manage.py runserver YOUR-PORT-NUMBER
``` 
### Accessed app via
```
localhost:8000
```
#### - or -
```
localhost:YOUR-PORT-NUMBER
```
